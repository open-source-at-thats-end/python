# Tech Stack
- Python
- FastAPI
- PyNest
- PyDantic
- Python dotenv
- SQL Alchemy (ORM)
- AioSqLite
- PyJWT
- JinJa2
- PoeThePoet
- StructLog
- Async implementation
- Swagger
- Strawberry Graphql
- CDN service
- DebugPy
- Poetry (package manager)
- uvicorn
- click (for CLI development)

# Purpose
- Provide REST API
- Provide GraphQL API
- Provide CLI interface development 
All in one app, single code base multiple interface.

# Support
It support any kind of Python development, which required to expose service through API.
